# 如何使用

1. 安裝依賴

```js
    npm install
```

2. 配置存放私鑰文件（data 文件夾）

```
    cd data
```

3. 選擇相應的命令運行相應的腳本
   - `npm run add-liquidity `
   - `npm run add-stake-lp`
   - `npm run cross-chain`

`npm run add-liquidity` 自動掃描 6 條鏈，將餘額大於 1USDT 而且不持有 LP 的添加 1U 到流動性池;

`npm run add-stake-lp` 自動掃描 6 條鏈，質押 LP;

`npm run cross-chain` 手動控制六條鏈之間互相跨鏈，每次金額 0 ～ 0.2U;
